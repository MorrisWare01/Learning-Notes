### C语言中的内存模型

* 栈区：存放局部变量、自动分配和释放（自顶向下进栈出栈）
* 堆区：动态内存分配、最大值为操作系统的80%
* 全局区：静态区
* 常量区：存放字符串
* 程序代码区：存放代码

### 动态分配内存

* 动态申请内存

```
//堆内存分配,40M
int* p = (int*)malloc(1024*1024*10*sizeof(int));
if(p == NULL){
    printf("内存申请失败!");
}
```

* 初始化内存

```
//将p指向的空间清0
memset(p,0,sizeof(int));    
```

* 重新分配内存

```
p=(int*)realloc(p,sizeof(int));
```

* 释放内存

```
//释放
if(p != NULL){
    free(p);
    p = NULL;
}
```
